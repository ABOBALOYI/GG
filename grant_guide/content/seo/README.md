This directory contains SEO-optimised content for Grant Guide SA. Each article is structured for search engine ranking and user helpfulness.

## Content Guidelines

All content follows these principles:
- South Africa-focused
- Simple, direct, practical language
- Structured for featured snippets
- Accurate and non-misleading
- Never promises funding approval
- Encourages verification on official funder websites

## Content Structure

Each article includes:

1. **SEO Metadata**
   - Title (max 60 chars)
   - Meta description (max 155 chars)
   - URL slug
   - Primary and secondary keywords
   - Search intent
   - Internal link suggestions
   - CTA suggestions

2. **Article Body**
   - H1 with primary keyword
   - Quick answer section (featured snippet)
   - Who qualifies section
   - Grants vs loans comparison
   - Where to find funding
   - Documents needed
   - Step-by-step application guide
   - Common mistakes
   - FAQs (minimum 6)
   - Next steps with CTAs

3. **Schema-Ready Block**
   - Article summary
   - Top 5 takeaways
   - FAQ schema pairs
